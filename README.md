# GROUP 1 APP PROJECT
## PAIN REFERRER
### Allows a quick look-up of potentially related diseases and illnesses associated with a subject's reported acute pain.

**App Flow:**
- User opens app
- User enters symptom
- User presses Search
- App loads CSV data
- App filters potential matches
- App displays results
- User can tap an item to Save it
